Time Value of
Money Calculator

An interactive tool to visualise how money grows over time through compounding and discounting.

📈
Compounding
Money grows forward in time. Interest earns interest each period.
FV = PV × (1 + i)ⁿ
📉
Discounting
A future amount is brought back to its value today.
PV = FV ÷ (1 + i)ⁿ

// How to use this calculator

  • 1
    Enter your four values in the input fields. PV = starting amount, FV = ending amount, i = interest rate per year, n = number of years.
  • 2
    Click "Calculate & Show Timeline" to generate the results.
  • 3
    Read the Formula Working section to see how PV and FV are calculated step by step.
  • 4
    Study the two timelines side by side. Green = Compounding (forward). Red = Discounting (backward). The year-by-year table shows the change each period.
  • 5
    Try different values to see how changing the rate or number of periods affects the outcome.
PV — Present Value ($)
FV — Future Value ($)
i — Interest Rate (%)
n — No. of Periods (years)
